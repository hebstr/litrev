# Feature Ideas

## Umbrella review / rapid review support

Review Type Adaptations lists systematic, meta-analysis, narrative, and scoping reviews. Two recognized synthesis types are missing:

- **Umbrella review** (review of reviews): different workflow — searches for systematic reviews/meta-analyses rather than primary studies. Would need adapted inclusion criteria and quality assessment (AMSTAR 2).
- **Rapid review**: streamlined version of a systematic review with fewer databases, simplified screening, and relaxed gates. Would need a defined subset of the current 7-phase workflow.
